🌍 Environmental Health PBL Website

This project is an interactive learning website based on Problem-Based Learning (PBL), focusing on environmental health issues such as:

* Water Quality 💧
* Air Pollution 🌫️
* Waste Management 🗑️
* Vector-Borne Diseases 🦟
* Climate Change & Health 🌍

🎯 Purpose

This website is designed to help students explore real-world problems, think critically, and develop solutions through interactive learning.
